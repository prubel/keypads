# keypads

[openscad](https://openscad.org/) files for generating 3d-printed keypads
along with [kmk](https://github.com/KMKfw/kmk_firmware) firmware files.


# 8key
Plans for a pico-pi-powered keypad with 8 keys in a 4x2 grid.
